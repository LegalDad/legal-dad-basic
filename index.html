Ваші проблеми і наші рішення

Блокування зв’язку

Не можете зв’язатися з дитиною через блоковані чати? Basic має готову заяву для оскарження.

Неправильні повістки

Отримали некоректну повістку? Basic відкриває доступ до шаблону заяви на скасування.

Страх пропустити термін

Переживаєте, що забудете подати заяву? Basic включає чек-ліст із термінами подачі.

Наші пакети

Basic (199 ₴)

  • 6 шаблонів заяв (DOCX + PDF)
  • Інструкція користувача (4 стор.)
  • Чек-ліст “Терміни подачі” (1 стор.)

Pro (499 ₴)

  • Усе з пакету Basic
  • Автозаповнення заяв через веб-форму
  • Перевірка статусу відправки
  • Telegram-бот для нагадувань (скоро)

Enterprise (999 ₴)

  • Усе з пакету Pro
  • Консультація юриста по Zoom
  • Підготовка індивідуальних документів
  • API-інтеграція для масових заяв

Електронні сервіси

  • Генератор заяв у браузері (DOCX + PDF)
  • Онлайн-оплата (LiqPay, Monobank Pay)
  • Подача заяв через API / веб-форми
  • Відстеження статусу звернення в Держреєстрі
  • Чат-бот для консультацій та нагадувань (скоро)
